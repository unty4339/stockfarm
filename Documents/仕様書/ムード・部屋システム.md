# ムード・部屋システム仕様書

牧場内の区画（部屋）と、部屋が持つムード数値の定義・効果を定めます。

## 1. 部屋システム

### 1.1 部屋の定義

柵・壁・扉によって完全に囲まれた閉じた領域を「部屋」として扱います。

### 1.2 部屋の判定ルール

- 柵・壁・扉のいずれかで全周囲が閉じている場合に部屋として認識されます
- 扉は開閉状態に関わらず、閉空間の判定においては壁と同等に扱われます
- フラッドフィル等のアルゴリズムで内部領域を特定し、外部と連通している場合は部屋として認識されません
- 入れ子構造（部屋の中に部屋）も有効で、それぞれ独立した部屋として扱われます

## 2. ムードシステム

### 2.1 概要

各部屋はムードと呼ばれる数値セットを持ちます。ムードは部屋内で活動する牛およびプレイヤーの作業効率・回復効率・繁殖率に影響します。

### 2.2 ムードの種類

| ムード種別 | 説明 | 主な効果 |
| --- | --- | --- |
| **作業ムード** | 作業への意欲・集中度 | 部屋内での作業効率が上昇する |
| **休憩ムード** | 休息環境の快適さ | 部屋内でのスタミナ回復速度が向上する |
| **繁殖ムード** | 交流・繁殖への意欲 | 部屋内での繁殖成功率が上昇する |

各ムードは独立した数値として管理されます。

### 2.3 ムードへの影響要因

#### 家具・設備による増減

- 部屋内に設備を配置することで、特定のムードが増加または減少します
- 各設備にはムード種別ごとの補正値（正・負）が設定されています（詳細は建築・設備システム仕様書を参照）
- 同一ムードへの補正は単純加算で合算されます
- ムード値の最大値は 100、最小値は -100 です

```
部屋の作業ムード = Σ（部屋内各設備の作業ムード補正値）
```

### 2.4 ムードの効果

#### 作業効率への影響

部屋の作業ムードは、現在地の牛・牧場主の作業効率計算において補正として参照されます。最終的な作業効率は次式で表されます。

$$E_{final} = \text{作業基本値} \times \frac{\text{幸福度}}{100} \times \left(1 + \frac{\text{作業ムード}}{100}\right)$$

- 各個体の作業基本値・幸福度の詳細は各個体システム仕様書を参照
- 牧場主は作業基本値を持たないため $E_{final} = \dfrac{\text{幸福度}}{100} \times \left(1 + \dfrac{\text{作業ムード}}{100}\right)$ となる
- 作業ムードが負の値の場合、効率が低下します

#### スタミナ回復への影響

部屋の休憩ムードは、睡眠中のスタミナ回復速度を補正します。

$$R_{room} = R_{base} \times \left(1 + \frac{\text{休憩ムード}}{100}\right)$$

- $R_{base}$: 睡眠中の1tickあたりの基本スタミナ回復量
- 休憩ムードが負の値の場合、回復速度が低下します

#### 繁殖率への影響

部屋の繁殖ムードは、繁殖行動の成功率を補正します。

$$P_{success} = \text{clamp}\left(P_{base} \times K_{estrus} \times \left(1 + \frac{\text{繁殖ムード}}{100}\right),\; 0,\; 1\right)$$

- $P_{base}$: 基本受胎確率（= 0.3）
- $K_{estrus}$: 発情期補正係数（「発情期」状態の牛の場合 ×2、通常 ×1）
- 発情期かつ繁殖ムード100の場合、成功率は clamp(0.3 × 2 × 2, 0, 1) = 100% となる
- 繁殖ムードが負の場合、成功率が低下します
