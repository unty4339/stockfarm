# UIシステム仕様書

プレイヤーが牧場を操作・管理するためのユーザーインターフェース全体の構成と挙動を定義します。

---

## 0. 用語定義

| 用語 | 定義 |
| --- | --- |
| **メインメニュー** | 画面下端に常時表示されるボタン列（「指令」「優先順位」「施設種別」など） |
| **メニュー要素** | メインメニューの各ボタンをクリックすると表示されるUI要素。スケジュール表・指令一覧・施設一覧など |
| **モード** | メニュー要素から選択され、マップ上のクリック挙動を上書きする状態。解体モード・施設設置モードなど |
| **ポップアップ** | ユニット・ゾーン・施設・アイテム等をクリックしたときに画面左下に表示される詳細データ |

---

## 1. 画面レイアウト概要

| 配置位置 | 要素 |
| --- | --- |
| **画面下端** | メインメニュー |
| **画面右下** | 時刻・速度表示エリア |
| **画面左下** | ポップアップ（施設・ワーカー・ゾーン・アイテム等をクリック時に表示） |

## 2. メインメニュー

### 2.1 構成

画面の最下端に固定表示されるボタン列で、左から以下の順に並びます。

```
[ 指令 ] [ 優先順位 ] [ 施設種別A ] [ 施設種別B ] [ 施設種別C ] ...
```

- 施設種別ボタンは、建築可能な施設のカテゴリー数だけ並びます（例: 建築・構造 / 柵・境界 / 給餌・リソース など）

### 2.2 メニュー要素の表示ルール

- いずれかのボタンをクリックすると、そのボタンの直上に対応する**メニュー要素**が表示されます
- メニュー要素は排他的に表示されます。あるメニュー要素を表示するとき、他のメニュー要素は非表示になります
- 別のボタンをクリックすると、表示中のメニュー要素を閉じて新しいものを表示します
- 同じボタンを再度クリックすると、メニュー要素を閉じます

### 2.3 モードのライフサイクル

- モードはメニュー要素から選択されることで設定されます
- モードは排他的に設定されます。あるモードに切り替わったときは既存のモードは解除されます
- メニュー要素が非表示になったタイミングで、そのメニュー要素から設定されたモードは解除されます
- 右クリックまたはEscキーを押すと現在のモードを解除します

## 3. メニュー要素詳細

### 3.1 [指令] メニュー要素

プレイヤーが施設や牛に対して直接操作を指示するためのメニュー要素です。各指令はボタンとして並んでいます。

#### 指令一覧

| 指令名 | 対象 | 操作 | モード | 効果 |
| --- | --- | --- | --- | --- |
| **解体** | 施設 | 左クリック | 解体モード | クリックした施設を即座に解体する（建築・設備システム §1 の撤去ルールに従い資金の50%を還元） |
| **種付け** | 牛 | 左クリック | 種付けモード | クリックした牛に種付けタスクを予約する |
| **搾乳** | 牛 | 左クリック | 搾乳モード | クリックした牛に搾乳タスクを予約する |

#### モードの挙動

- 指令ボタンをクリックすると対応するモードが設定されます
- モードが設定された状態では、対象（施設または牛）を左クリックすることで指令が実行されます
- 別の指令ボタンをクリックすると、モードが切り替わります（§2.3 参照）

### 3.2 [優先順位] メニュー要素

牧場主と牛それぞれのタスク優先度を一覧で確認・編集できる表形式のメニュー要素です。

#### レイアウト

| （名前列） | タスク種別1 | タスク種別2 | … |
| --- | --- | --- | --- |
| 牧場主A | 3 | 1 | … |
| 牛B | 2 | 5 | … |
| … | … | … | … |

- **行**: 牧場主・牛が1エンティティにつき1行
- **列**: 名前列 + タスク種別ごとの優先度列
- **セル**: 優先度を表す整数値が表示される

#### セルの操作

| 操作 | 結果 |
| --- | --- |
| 左クリック | 優先度の値を +1 |
| 右クリック | 優先度の値を −1 |

- 優先度の最小値・最大値は行動AIロジックの定義に従います

### 3.3 [施設種別] メニュー要素

クリックした施設種別カテゴリーに属する施設の一覧が表示されます。

- 各施設はボタンとして並んでいます
- ボタンをクリックすると**施設設置モード**に移行します（§7 参照）

## 4. 施設ポップアップ

### 4.1 表示トリガー

施設ポップアップは以下のいずれかの操作で表示されます。いずれも、モードが設定されていない状態が前提です。

| 操作 | 条件 | 結果 |
| --- | --- | --- |
| マップ上の施設を**左クリック** | — | その施設のポップアップを表示 |
| マップ上を**左クリック・ドラッグ**してドロップ | ドラッグ範囲内にワーカーが0人かつ施設が1つ | その施設のポップアップを表示 |
| マップ上を**左クリック・ドラッグ**してドロップ | ドラッグ範囲内にワーカーが0人かつ施設が複数 | 複数施設アイコンバーを表示（§4.4 参照） |

- ドラッグ範囲内にワーカーが1人以上いる場合はワーカー選択が優先され、施設選択は行いません（§5 参照）
- ドラッグ中は半透明の矩形が画面上に表示され、選択範囲を視覚的に示します

### 4.2 表示内容

| 要素 | 説明 |
| --- | --- |
| **施設画像** | 対象施設のアイコンまたはスプライト |
| **施設名称** | 施設の名称テキスト |
| **アサイン枠** | 施設に割り当てられた牧場主・牛のスロット。アサイン枠が複数ある施設では枠を複数表示する |
| **セットされたアイテム** | 施設に投入・セットされている固体アイテムの一覧 |
| **プログレスバー** | 現在進行中の処理の進捗を示すバー |
| **チェスト内容** | 内蔵チェストが併設されている場合、格納中のアイテムの一覧を表示 |
| **タンク内容** | 内蔵液体タンクが併設されている場合、現在の液体量・液体種別・品質を表示 |

- 併設されているチェスト・タンクがない施設では、それぞれの行は表示しません

### 4.3 閉じる

- ポップアップ外を左クリックするとポップアップを閉じます

### 4.4 複数施設選択時のアイコンバー

矩形ドラッグの範囲内に複数の施設が含まれており、かつワーカーが0人だった場合、画面下部にその施設たちのアイコンバーを表示します。

| 要素 | 説明 |
| --- | --- |
| **施設アイコン** | 施設ごとのアイコンボタン（施設色・種別名を表示） |

- アイコンのいずれかをクリックすると、その施設のポップアップを表示します（§4.1 の結果と同じ）
- アイコンバーは空白エリアをクリックすると閉じます

## 5. ワーカーポップアップ

### 5.1 表示トリガー

ワーカーポップアップは以下のいずれかの操作で表示されます。いずれも、モードが設定されていない状態が前提です。

| 操作 | 条件 | 結果 |
| --- | --- | --- |
| マップ上の牧場主または牛を**左クリック** | — | そのワーカーのポップアップを表示 |
| マップ上を**左クリック・ドラッグ**してドロップ | ドラッグ範囲内にワーカーが1人 | そのワーカーのポップアップを表示 |
| マップ上を**左クリック・ドラッグ**してドロップ | ドラッグ範囲内にワーカーが複数 | 複数ワーカーアイコンバーを表示（§5.4 参照） |

ドラッグ中は半透明の矩形が画面上に表示され、選択範囲を視覚的に示します。

### 5.2 表示内容

| 要素 | 説明 |
| --- | --- |
| **ワーカー画像** | 対象の牧場主または牛のアイコンまたはスプライト |
| **ワーカー名称** | 牧場主または牛の名称テキスト |
| **現在のタスク** | 現在実行中または待機中のタスク種別 |
| **現在のステータス** | 空腹度・スタミナ・ムードなど、対象の現在の状態値 |

### 5.3 閉じる

- ポップアップ外の空白エリアを左クリックするとポップアップを閉じます

### 5.4 複数ワーカー選択時のアイコンバー

矩形ドラッグの範囲内に複数のワーカーが含まれていた場合、画面下部にそのワーカーたちのアイコンバーを表示します。

| 要素 | 説明 |
| --- | --- |
| **ワーカーアイコン** | ワーカーごとのアイコンボタン（ワーカー色・名称を表示） |

- アイコンのいずれかをクリックすると、そのワーカーのポップアップを表示します（§5.1 の結果と同じ）
- アイコンバーは空白エリアをクリックすると閉じます

## 7. 施設設置モード

### 7.1 概要

施設種別メニュー要素で施設ボタンをクリックすると、施設設置モードへ移行します。このモードは建築・設備システム仕様書 §1 の「仮設置（ゴースト）機能」に対応します。

施設種別メニュー要素が閉じられると施設設置モードは自動的に解除されます（§2.3 参照）。

### 7.2 カーソル操作

| 操作 | 結果 |
| --- | --- |
| マップ上にカーソルを置く | カーソル位置のグリッドに、選択した施設のゴースト（半透明プレビュー）を表示 |
| 左クリック | カーソル位置のグリッドに施設を配置（資金が足りない場合は配置不可） |
| 右クリック / Escキー | 施設設置モードを終了してゴーストを消去 |

- ゴーストは配置可能なグリッドでは通常色（半透明）、配置不可なグリッドでは赤みがかった表示になります
- 配置不可の条件（他の建築物との重複、範囲外など）は建築・設備システム仕様書 §1 に従います

---

## 8. 右下：時刻・速度表示エリア

画面右下に常時表示される情報エリアです。

### 8.1 表示内容

| 要素 | 内容 |
| --- | --- |
| **経過日数** | ゲーム開始からの日数（例: `Day 42`） |
| **現在時刻** | ゲーム内の現在時刻（例: `12:30`） |
| **時間倍率ボタン** | 現在の速度倍率を表示し、クリックで切り替えるボタン |

### 8.2 時間倍率ボタン

- 表示例: `×1` `×2` `×3`
- クリックするたびに倍率が順に切り替わります（×1 → ×2 → ×3 → ×1 …）
- ゲーム内時間の倍率定義はゲーム内時間仕様（牧場マップ・経済システム §0.1）に従います
