# 牛の個体システム仕様書

本作の牛は、プレイヤー（飼育員）と協力して牧場を運営するパートナーです。複雑な生存リスクを抑え、行動の最適化と作業分担、次世代への命の継承（種付け・繁殖）、そして「状態異常」による個体ごとの変化にフォーカスした設計としています。

登場する牛はすべて雌です。繁殖はプレイヤーが「種付け施設」を操作することで行います。

## 1. ステータス構成

### 1.1 特徴ステータス（固定・基本値）

個体ごとに生成時に決定される、その牛の基礎能力です。

- **ID**: 個体を識別するための一意の番号。
- **個体名**: プレイヤーが命名、または自動生成される名前。
- **年齢**: 誕生からの経過日数。ライフサイクル（子牛、若牛など）の判定基準。
- **運動基本値**: 移動速度の基礎となる値。
- **作業基本値**: 運搬や修理などの全作業における効率の基礎となる値。
- **乳質基本値**: 生産される牛乳の品質の基礎となる値。

#### 子牛誕生時の個体値生成

子牛の各基本値は母牛のステータスを基準に決定されます。

$$\text{基本値} = \text{clamp}\left(\text{母の値} + \text{乱数}(-10, +10) + \frac{\text{出産時の母の幸福度}}{10},\; 1,\; 100\right)$$

- 母の基本値を基準に ±10 の乱数変動が加わる
- 出産時の母の幸福度が高いほど最大 +10 のボーナスが乗る（幸福度100で +10）
- 最終値は 1〜100 にクランプされる

### 1.2 動的ステータス（0〜100）

ゲームプレイ中に常に変動する数値です。

- **空腹度 (Hunger)**: 時間経過で減少（**80 ポイント / 日**）。0になると「空腹」状態異常が発生し、スタミナが急速に減少し始める。食事1回につき空腹度が **80** 回復する。
- **スタミナ (Stamina)**: 行動や作業によって減少。0になるとその場で「強制睡眠」に入る。詳細は §6.3 を参照。
- **乳蓄積量 (Milk Level)**: 状態異常「泌乳期」の個体のみ増加（**100 ポイント / 日**、1日1回搾乳可能になる）。100で「搾乳可能」。
- **妊娠経過 (Pregnancy Progress)**: 状態異常「妊娠」の個体のみ増加（**20 ポイント / 日** → **5日で出産**）。100で出産。
- **幸福度 (Happiness)**: 環境やケアで変動。作業効率、品質、種付け成功率に影響。

### 1.3 幸福度の変化要因

| 要因 | 変化量（1日あたり） |
| --- | --- |
| 食事摂取 | +5 |
| 「空腹」状態が続く | -10 |
| ケア利用（自動ブラシ機） | +15 |
| 「疲労」状態で行動 | -5 |
| 「栄養失調」状態 | -10 |
| 廃棄物マーカーが隣接タイルにある | -5 |
| 休憩ムード50以上の部屋で睡眠 | +10 |

## 2. 状態異常システム (Status Conditions)

牛は複数の状態異常を重複して持つことができます。これらはパラメータへの補正や、見た目を変えるフラグとして機能します。

| 状態名 | 発生条件 | 効果・補正 | 見た目の変化 |
| --- | --- | --- | --- |
| **妊娠（前期）** | 種付け成功後 (Progress 1-50) | 空腹消費 +20% | 特になし |
| **妊娠（後期）** | 妊娠経過 51-100 | 空腹消費 +50% / 移動速度 -30% / スタミナ消費 +30% | お腹が大きく膨らむ |
| **泌乳期** | 出産後 30 日間 | 乳蓄積が有効化 / 空腹消費 +30% | 乳房が張る |
| **疲労** | スタミナ 20以下 | 作業効率 -50% / 移動速度 -20% | 息を切らしている動作 |
| **発情期** | 非妊娠の成牛が対象 / 3〜7日ごとにランダム発生 / 継続2日間 | 種付け成功率アップ | ハートのエフェクト |
| **空腹** | 空腹度が 0 | スタミナ消費速度 +30% | 空腹のエフェクト |
| **栄養失調** | 空腹度 0 が 1日以上継続 | スタミナ最大値 -50% | やつれた見た目 |

## 3. ライフサイクル

死亡の概念はなく、年齢や状態異常によって役割が変化します。すべての牛は雌です。

| 段階 | 移行条件 | 能力 |
| --- | --- | --- |
| **子牛 (Calf)** | 誕生〜29日 | 作業不可、種付け不可 |
| **若牛 (Heifer)** | 30日以上（かつ未出産） | 作業解禁、種付け可能 |
| **成牛 (Cow)** | 初めての出産で移行 | 泌乳期に搾乳可能 |

## 4. 繁殖システム

### 4.1 種付けの手順

1. プレイヤーが「種付け施設」へ移動する。
2. プレイヤーが種付けしたい牛を指定して施設へ誘導する（強制タスクとして牛に送られる）。
3. 牛と牧場主が施設内の各スロットに入った状態でプレイヤーが操作を実行すると、種付けが完了する。
4. 成功すると対象の牛に「妊娠（前期）」状態異常が付与される。

### 4.2 種付けの成功率

種付けの成否は確率で判定されます。

$$P_{success} = \text{clamp}\left(P_{base} \times K_{estrus} \times \left(1 + \frac{\text{繁殖ムード}}{100}\right),\; 0,\; 1\right)$$

- $P_{base}$: 基本成功率 = 0.3（30%）
- $K_{estrus}$: 発情期補正係数（「発情期」状態の牛の場合 ×2、通常 ×1）
- 繁殖ムードは種付け施設が配置されている部屋の値を参照する（詳細はムード・部屋システム仕様書参照）

### 4.3 出産

- 「妊娠経過」が 100 に達すると、現在地またはベッドの上で出産し子牛が生成される
- 出産後、母牛は「泌乳期」状態異常を 30 日間持つ

## 5. 作業管理システム

牛ごとに「どのカテゴリーの作業を許可するか」をプレイヤーが設定できます。
作業の実行速度や移動には「作業基本値」「運動基本値」が参照されます。

- **運搬 (Hauling)**: 収穫物や資材の運搬。
- **加工補助 (Processing)**: 設備の動力（足踏み等）。
- **清掃・整理 (Organization)**: 牧場内の糞尿・廃棄物の撤去。

## 6. 計算式

### 6.1 乳質・成果物品質 ($Q$)

$$Q = \text{乳質基本値} \times \frac{\text{幸福度}}{100} \times \text{餌の質係数}$$

- 「栄養失調」状態異常によりさらにマイナス補正がかかる（係数 ×0.5）

### 6.2 作業効率 ($E$)

$$E = \text{作業基本値} \times \frac{\text{幸福度}}{100} \times \left(1 + \frac{\text{作業ムード}}{100}\right)$$

- 「妊娠（後期）」「疲労」などの状態異常により大幅に低下する
- 作業ムードは現在地の部屋の値を参照する（詳細はムード・部屋システム仕様書を参照）

### 6.3 スタミナ変化速度

| 状況 | スタミナ変化量 |
| --- | --- |
| 睡眠中 | +0.4 / tick |
| 加工補助・清掃中 | -0.2 / tick |
| 移動中 | -0.1 / tick |
| アイドル時 | -0.05 / tick |

- 睡眠完了条件はスタミナが100になるか、就寝スケジュールが終了するかのいずれか早い方
