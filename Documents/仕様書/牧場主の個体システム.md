# 牧場主の個体システム仕様書

牧場主はプレイヤーが直接操作するキャラクターです。牛と同様の基本的な生理ニーズを持ちつつ、牧場全体の管理・指示・直接作業を担います。

## 1. 動的ステータス（0〜100）

ゲームプレイ中に常に変動する数値です。

- **空腹度 (Hunger)**: 時間経過で減少（**80 ポイント / 日**）。0になると「スタミナ」が急速に減少し始める。食事1回につき空腹度が **80** 回復する。
- **スタミナ (Stamina)**: 行動や作業によって減少。0になるとその場で「強制睡眠」に入る。
- **幸福度 (Happiness)**: 環境やケアで変動。作業効率に影響。

### 1.1 幸福度の変化要因

| 要因 | 変化量（1日あたり） |
| --- | --- |
| 休憩ムード50以上の部屋で睡眠 | +10 |
| 食事摂取 | +5 |
| 「疲労」状態で行動 | -5 |
| 「栄養失調」状態 | -10 |
| 作業ムード50以上の部屋で作業 | +5 |

## 2. 状態異常

| 状態名 | 発生条件 | 効果・補正 |
| --- | --- | --- |
| **疲労** | スタミナ 20以下 | 作業効率 -50% / 移動速度 -20% |
| **栄養失調** | 空腹度 0 が 1日以上継続 | スタミナ最大値 -50% |

## 3. 計算式

### 3.1 作業効率 ($E$)

$$E = \frac{\text{幸福度}}{100} \times \left(1 + \frac{\text{作業ムード}}{100}\right)$$

- 「疲労」状態によりさらに低下する（係数 ×0.5）
- 作業ムードは現在地の部屋の値を参照する（詳細はムード・部屋システム仕様書を参照）

### 3.2 スタミナ変化速度

| 状況 | スタミナ変化量 |
| --- | --- |
| 睡眠中 | +0.4 / tick |
| 直接作業中 | -0.2 / tick |
| 移動中 | -0.1 / tick |
| アイドル時 | -0.05 / tick |

- 睡眠完了条件はスタミナが100になるか、就寝スケジュールが終了するかのいずれか早い方

## 4. 睡眠とベッド割り当て

牧場主の睡眠挙動は牛のベッド割り当てシステムと同じルールに従う（詳細は建築・設備システム仕様書 §3.4 参照）。

- スタミナが低下するか就寝時間になると睡眠を開始する
- 自身に割り当てられたベッドへ向かい、そこで眠る
- 割り当てベッドがない場合、未割り当ての空きベッドを探して紐づけ、眠る
- 空きベッドもない場合、現在地でそのまま眠る
- ベッドをクリックして割り当てを手動変更・解除できる（牛・牧場主の割り当てを区別しない）

## 5. 活動不能状態の挙動

空腹度が0で「栄養失調」になると、睡眠と覚醒を繰り返す活動不能ループに陥ります。この状態を解消するには、食料を摂取する必要があります。

## 6. 食料摂取システム

牧場主は「食料棚」設備から食料を取り出して食べることで空腹度を回復します。

- **食料棚**: 1×1サイズの設備。食料アイテムを格納し、牧場主がアクセスして摂取できる
- 食料棚に格納できる食料は牛の飼料と同じ「栄養ペレット」を使用する（専用食料なし）
- 食料は「資材の発注」と同じく車に発注することで補充できる
- 食事1回につき空腹度が 80 回復する
