# 牧場主の個体システム仕様書

牧場主はプレイヤーが直接操作するキャラクターです。牛と同様の基本的な生理ニーズを持ちつつ、牧場全体の管理・指示・直接作業を担います。

## 1. 動的ステータス（0〜100）

ゲームプレイ中に常に変動する数値です。

- **空腹度 (Hunger)**: 時間経過で減少。0になると「スタミナ」が急速に減少し始める。
- **スタミナ (Stamina)**: 行動や作業によって減少。0になるとその場で「強制睡眠」に入る。
- **幸福度 (Happiness)**: 環境やケアで変動。作業効率に影響。

## 2. 状態異常

| 状態名 | 発生条件 | 効果・補正 |
| --- | --- | --- |
| **疲労** | スタミナ 20以下 | 作業効率 -50% / 移動速度 -20% |
| **栄養失調** | 空腹度 0 が長時間継続 | スタミナ最大値 -50% |

## 3. 計算式

### 3.1 作業効率 ($E$)

$$E = \frac{\text{幸福度}}{100}$$

※「疲労」状態によりさらに低下する。

## 4. 活動不能状態の挙動

空腹度が0で「栄養失調」になると、睡眠と覚醒を繰り返す活動不能ループに陥ります。この状態を解消するには、食料を摂取する必要があります。
