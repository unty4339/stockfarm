# 牧場マップ・経済システム仕様書

牧場の地理的な構造と、売却を通じた資金獲得の仕組みを定義します。

## 0. ゲーム概要

- **ジャンル**: 牧場経営シミュレーション（サンドボックス型）
- **視点**: 2D 見下ろし（Top-down）
- **目標**: 明確なクリア条件はなく、牧場を自由に拡張・発展させ続けることができる

## 0.1 ゲーム内時間

| 設定 | 値 |
| --- | --- |
| 1ゲーム内日 | 600 tick |
| 1倍速時の換算 | 1 tick = 1 秒（1日 = 10 分） |
| ゲーム速度倍率 | プレイヤーがゲーム中に変更可能（×1〜×3） |
| tick/秒 倍率 | ゲーム設定から個別に変更可能 |

## 1. 資金システム

### 1.1 概要

資金は牧場の建設・拡張・購買に使用する基本的な経済リソースです。アイテムを売却することで入手できます。

### 1.2 資金の入手方法

牧場内で生産した固体リソースを売却用の資源置き場に運び込んでおくと、定期的な車の回収時に資金へ変換されます。

### 1.3 売却価格テーブル

| アイテム | 基準価格 | 品質補正 |
| --- | --- | --- |
| **瓶牛乳** | 40 資金 | × (品質 / 100) |
| **バター** | 80 資金 | × (品質 / 100) |
| **チーズ** | 120 資金 | × (品質 / 100) |

- 品質 100 のアイテムが基準価格で売れ、品質が低下するほど比例して価格が下がる

## 2. 売却システム

### 2.1 道路と資源置き場

- 牧場の東端は道路に面しており、この辺は変更・拡張ができません
- 道路沿いに売却用の資源置き場が設置されています
- 資源置き場はチェストと同様に扱われます。優先度・フィルター条件を設定でき、自動運搬タスクによってアイテムを搬入できます
- 資源置き場に固体リソースを搬入しておくと、次回の回収対象になります

### 2.2 回収スケジュール

- 1ゲーム内日に2回、車が来て資源置き場のリソースをすべて回収します
- 回収されたリソースはアイテム種別ごとの売却価格に従い、即座に資金へ変換されます
- 回収後、資源置き場は空になります

## 3. マップシステム

### 3.1 牧場の初期サイズ

- グリッドサイズ: 30×30マス
- 東端: 道路に面しており、拡張不可

### 3.2 マップ拡張

- 南・北・西の3方向に拡張できます
- 各方向について、資金を支払うことで1回あたり15マス追加されます
- 同一方向への複数回拡張も可能で、都度費用が必要です
- 拡張費用は拡張回数（方向ごとの累計）に応じて増加します

| 拡張回数（方向ごと） | 費用 |
| --- | --- |
| 1回目 | 500 資金 |
| 2回目 | 1,500 資金 |
| 3回目 | 4,500 資金 |
| N回目 | 500 × 3^(N-1) 資金 |

### 3.3 ゲーム開始時の初期状態

ゲーム開始直後の牧場の状態は以下の通りです。

| 項目 | 内容 |
| --- | --- |
| **初期資金** | 2,000 資金 |
| **初期の牛** | 成牛2頭（雌2頭） |
| **初期設備** | 給餌桶 ×2、チェスト ×2 |
| **初期囲い** | 柵で仕切られた 8×8 マスのエリア（牛の初期居住区） |
