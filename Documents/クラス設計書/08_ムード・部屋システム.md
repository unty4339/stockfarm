## RoomData

```
class（MonoBehaviourではない、データオブジェクト）
```

柵・壁・ゲートで囲まれた閉領域（部屋）を表すデータクラス。

### プロパティ

| 名前 | 型 | 説明 |
|---|---|---|
| `TilePositions` | `HashSet<Vector2Int>` | 部屋内タイル座標の集合 |
| `ContainedEquipments` | `List<EquipmentBase>` | 部屋内に配置された設備の一覧 |
| `CurrentMood` | `MoodData` | 現在のムード値（MoodCalculatorで計算） |
| `WasteMarkers` | `List<WasteMarker>` | 部屋内の廃棄物マーカー一覧 |

### メソッド

```csharp
// 指定座標が部屋内に含まれるかを返す
bool Contains(Vector2Int position)

// 部屋内に指定タイプの設備が存在するかを返す
bool HasEquipment(EquipmentType type)

// 部屋内の指定タイプの設備を全て返す
List<EquipmentBase> GetEquipments(EquipmentType type)
```

## RoomDetector (static)

フラッドフィルアルゴリズムを使い、グリッドタイルから部屋領域を検出するクラス。

```csharp
static class RoomDetector
{
    // グリッド全体を走査し、全部屋のRoomDataリストを返す
    // 壁・柵・ゲートを境界として床・地面タイルの連続領域を部屋とみなす
    static List<RoomData> DetectRooms(GridTile[,] grid)

    // 指定座標を含む部屋をリストから検索して返す（見つからなければnull）
    static RoomData GetRoomAt(Vector2Int position, List<RoomData> rooms)
}
```

### 部屋判定のルール

- 柵（Fence）・壁（Wall）・ゲート（Gate）で完全に囲まれた床・地面タイルの連続領域を1つの部屋とする
- ゲートは境界として機能するが、通路として通行可能
- 部屋として認識されるには最低1タイルの床・地面が必要

## MoodCalculator (static)

部屋内の設備配置と廃棄物マーカーに基づきムード値を計算するクラス。

```csharp
static class MoodCalculator
{
    // 部屋内の設備ボーナス合計から廃棄物ペナルティ合計を差し引いたMoodDataを返す
    // 各ムード値は 0〜100 にclampする
    static MoodData Calculate(RoomData room)

    // 作業効率を返す
    // result = Happiness/100 × (1 + workMood/100)
    static float GetWorkEfficiency(float happiness, float workMood)

    // スタミナ回復速度を返す
    // result = baseRate × bedCoefficient × (1 + restMood/100)
    static float GetStaminaRecoveryRate(float baseRate, float bedCoefficient, float restMood)

    // 繁殖成功率を返す（0〜1にclamp）
    // result = clamp(baseProbability × estrusFactor × (1 + breedMood/100), 0, 1)
    static float GetBreedSuccessRate(float baseProbability, float estrusFactor, float breedMood)
}
```

## RoomManager : MonoBehaviour (Singleton)

部屋の検出結果を保持し、ワーカーへのムード値提供を担うクラス。  
設備の設置・撤去のたびに`DetectRooms()`を再実行して部屋情報を更新する。

### プロパティ

| 名前 | 型 | 説明 |
|---|---|---|
| `Rooms` | `List<RoomData>` | 現在検出されている全部屋のリスト |

### メソッド

```csharp
// グリッド全体を再走査して部屋リストを更新する
// BuildingManagerの設備変更後に呼ぶ
void RefreshRooms()

// 指定座標が属する部屋のMoodDataを返す（部屋なしの場合はデフォルト値）
MoodData GetMoodAt(Vector2Int position)

// 指定座標が何らかの部屋に属しているかを返す
bool IsInsideRoom(Vector2Int position)
```
