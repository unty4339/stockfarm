## ResourceBase

```
abstract class（MonoBehaviourではない、データオブジェクト）
```

全リソースの基底クラス。

### プロパティ

| 名前 | 型 | 説明 |
|---|---|---|
| `Type` | `ResourceType` | リソース種別 |
| `Amount` | `int` | 数量 |

## ResourceType (enum)

```csharp
enum ResourceType
{
    NutritionPellet,  // 栄養ペレット（固体・スタック不可）
    Cheese,           // チーズ（固体・スタック不可・品質あり）
    Butter,           // バター（固体・スタック不可・品質あり）
    BottledMilk,      // 瓶牛乳（固体・スタック不可・品質あり）
    Milk,             // 牛乳（液体）
    Wheat,            // 小麦（固体・スタック可能・上限50）
}
```

## TileItem

```
class（MonoBehaviourではない、データオブジェクト）
```

タイル上に配置される1アイテムスロットを表すクラス。  
スタック可否・スタック上限は `ResourceType` によって静的に決まる。

### プロパティ

| 名前 | 型 | 説明 |
|---|---|---|
| `Type` | `ResourceType` | アイテム種別 |
| `StackCount` | `int` | 現在のスタック数（非スタックアイテムは常に1） |
| `Quality` | `float?` | 品質（品質ありアイテムのみ非null） |
| `IsStackable` | `bool` | スタック可能かどうか |
| `StackLimit` | `int` | スタック上限（非スタックアイテムは0） |
| `IsStackFull` | `bool` | スタックが上限に達しているか |

```csharp
// 品質あり（非スタック）アイテム用コンストラクタ
TileItem(ResourceType type, float quality)

// スタック可能アイテム用コンストラクタ
TileItem(ResourceType type, int stackCount = 1)

// 指定数をスタックに追加し、追加できなかった余剰数を返す
int TryAddStack(int count)

// ResourceTypeごとのスタック上限を返す（0=非スタック可）
static int GetStackLimit(ResourceType type)
```

## ItemDropService (static)

アイテムをタイルに落とす処理を担う静的クラス。

```csharp
// 指定座標にアイテムを落とす
// 配置できない場合は時計回りに隣接8タイルを探索して最初の空きタイルに落とす
static void DropItem(TileItem item, Vector2Int targetPosition)
```

**時計回り8方向順**: 上 → 右上 → 右 → 右下 → 下 → 左下 → 左 → 左上

## LiquidResource : ResourceBase

液体リソース（牛乳）を表すクラス。品質値を持つ。

### プロパティ

| 名前 | 型 | 説明 |
|---|---|---|
| `Quality` | `float` | 品質（0〜100） |

```csharp
// コンストラクタ
LiquidResource(int amount, float quality)

// 別の液体リソースと混合する
// 混合後の品質 = (既存量×既存品質 + other.Amount×other.Quality) / (既存量 + other.Amount)
// 混合後の数量 = 既存量 + other.Amount
void Mix(LiquidResource other)
```

## IContainer (interface)

リソースを格納・取り出し可能なコンテナのインタフェース。  
ChestとLiquidTankが実装する。

```csharp
interface IContainer
{
    // リソースを格納する（容量超過・型不一致の場合はfalseを返す）
    bool TryStore(ResourceBase resource)

    // 指定種別・数量のリソースを取り出す（不足の場合はfalseを返す）
    bool TryTake(ResourceType type, int amount, out ResourceBase resource)

    // 指定種別の現在保管数量を返す
    int GetAmount(ResourceType type)

    // 空き容量を返す
    int GetRemainingCapacity()
}
```

## Chest : EquipmentBase, IContainer

固体リソース専用の保管設備。

## WasteMarker

廃棄物（糞尿）のタイルマーカーを表すデータクラス。  
MonoBehaviourではなく、GridTileに格納される純粋なデータオブジェクト。

### プロパティ

| 名前 | 型 | 説明 |
|---|---|---|
| `TilePosition` | `Vector2Int` | マーカーが置かれたタイル座標 |
| `Severity` | `int` | 深刻度（ムード低下量に使用、0で消滅） |

### メソッド

```csharp
// tickごとに時間経過処理を行う（Severityをデクリメント）
// 清掃タスクによっても呼ばれる
void Decay(int amount)
```
