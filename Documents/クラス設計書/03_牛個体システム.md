## CowWorker

```
MonoBehaviour : WorkerBase
```

牛の個体を表すクラス。WorkerBaseの共通ステータスに加え、乳蓄積・繁殖・ライフサイクル管理を担う。

### 特徴ステータス（固定値）

| 名前 | 型 | 説明 |
|---|---|---|
| `CowId` | `string` | 個体識別ID（ゲーム内ユニーク） |
| `Name` | `string` | 個体名 |
| `BaseMovement` | `float` | 移動基本値（経路コスト計算に使用） |
| `BaseWork` | `float` | 作業基本値（加工補助効率の基礎） |
| `BaseMilkQuality` | `float` | 乳質基本値（0〜100） |

### 動的ステータス

| 名前 | 型 | 説明 |
|---|---|---|
| `AgeInDays` | `int` | 年齢（日数） |
| `LifecycleStage` | `CowLifecycleStage` | ライフサイクル段階 |
| `MilkAccumulation` | `float` | 現在の乳蓄積量（搾乳で0にリセット） |
| `PregnancyProgress` | `int` | 妊娠経過tick数 |

### 乳蓄積メソッド

```csharp
// tickごとに乳を蓄積する（泌乳期のみ有効）
// 蓄積量 = 基本蓄積レート × 作業効率
void AccumulateMilk(float workMood)

// 搾乳を実行し、蓄積量に応じたLiquidResourceを返す
// 搾乳後MilkAccumulationは0にリセットされる
LiquidResource Milk()

// 現在の乳質を返す
// 乳質 = BaseMilkQuality × (Happiness / 100)
float GetMilkQuality()
```

### 繁殖メソッド

```csharp
// 妊娠を開始する（StatusEffectType.PregnancyEarlyを付与）
// 呼び出し前に繁殖可否チェックを済ませておくこと
void StartPregnancy()

// tickごとに妊娠を進める
// 前期→後期の移行、出産タイミングを内部で管理する
void AdvancePregnancy()

// 出産処理を実行する
// 新しいCowWorkerインスタンスの生成はGameManagerなど外部が担う
// 本メソッドは自身の状態のみをリセットする
void GiveBirth()

// 繁殖成功率を返す
// rate = clamp(基本確率 × 発情期係数 × (1 + breedMood/100), 0, 1)
// Estrusを持っていない場合は基本確率のみを使用する
float GetBreedSuccessRate(float breedMood)
```

### ライフサイクルメソッド

```csharp
// 1日経過処理（AgeInDaysをインクリメント、LifecycleStageを更新）
void AgeUp()
```

## CowLifecycleStage (enum)

```csharp
enum CowLifecycleStage
{
    Calf,       // 子牛（AgeInDays 0〜29）
    YoungCow,   // 若牛（AgeInDays 30以上、未出産）
    AdultCow,   // 成牛（出産経験あり）
}
```

### 段階移行の条件

| 遷移 | 条件 |
|---|---|
| Calf → YoungCow | AgeInDays が 30 に達した時 |
| YoungCow → AdultCow | 初回出産（GiveBirth()呼び出し）時 |

### 各段階の制約

| 段階 | 繁殖 | 搾乳 | 乳蓄積 |
|---|---|---|---|
| Calf | 不可 | 不可 | しない |
| YoungCow | 可 | 不可 | しない |
| AdultCow | 可 | 可 | Lactating状態のみ |
