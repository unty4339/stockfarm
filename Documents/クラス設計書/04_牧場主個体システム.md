## FarmerWorker

```
MonoBehaviour : WorkerBase
```

プレイヤーが操作する牧場主の個体クラス。  
WorkerBaseの共通ステータスに加え、睡眠・摂食の特殊処理を持つ。

牛との違い：
- 繁殖・乳蓄積の機能を持たない
- 栄養失調で睡眠・覚醒ループによる活動不能状態になる

### 追加プロパティ

| 名前 | 型 | 説明 |
|---|---|---|
| `AssignedBed` | `CowBed` | 割り当てられたベッド（nullなら未割り当て） |
| `IsIncapacitated` | `bool` | 栄養失調による活動不能状態フラグ |

### メソッド

```csharp
// 睡眠を実行する
// restMood >= 50 の場合、Happiness += 10
// スタミナを GetStaminaRecoveryRate() に基づき回復する
void Sleep(float restMood)

// 食料棚から摂食する
// Hunger -= 80（0にclamp）
// Malnutrition / Hungry 状態異常を解除する
void Eat(FoodShelf shelf)

// 作業効率を返す（WorkerBaseをoverride）
// rate = Happiness/100 × (1 + workMood/100)
override float GetWorkEfficiency(float workMood)
```

### 活動不能状態の条件

Malnutrition 状態異常が付与されると `IsIncapacitated = true` になり、  
AIが睡眠タスクと摂食タスクのみを繰り返す状態（睡眠・覚醒ループ）に入る。  
摂食によりMalnutritionが解除されると `IsIncapacitated = false` に戻る。

### 幸福度の変化要因

| 要因 | 変化量 | 条件 |
|---|---|---|
| 睡眠 | +10 / 睡眠 | restMood ≥ 50 |
| 食事 | +5 / 食事 | - |
| 疲労（Fatigue） | -5 / tick | Fatigue状態中 |
| 栄養失調（Malnutrition） | -10 / tick | Malnutrition状態中 |
| 廃棄物マーカー | -Severity / tick | 同タイル上に存在 |
| 作業完了 | +5 / 完了 | 作業ムード ≥ 50 |
