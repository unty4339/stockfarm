## MenuBarUI : MonoBehaviour

画面下端のメインメニューバー。3つのサブメニューを排他表示で管理する。

### プロパティ

| 名前 | 型 | 説明 |
|---|---|---|
| `CommandMenuUI` | `CommandMenuUI` | 指令メニューの参照 |
| `PriorityMenuUI` | `PriorityMenuUI` | 優先順位メニューの参照 |
| `FacilityMenuUI` | `FacilityMenuUI` | 施設種別メニューの参照 |

### メソッド

```csharp
// 指令メニューを表示する（他メニューは非表示にする）
void ShowCommandMenu()

// 優先順位メニューを表示する（他メニューは非表示にする）
void ShowPriorityMenu()

// 施設種別メニューを表示する（他メニューは非表示にする）
void ShowFacilityMenu()

// 全サブメニューを非表示にする
void HideAll()
```

## CommandMenuUI : MonoBehaviour

指令メニュー。指定タスクをプレイヤーが強制指示するためのUI。

### メソッド

```csharp
// 解体指令ボタン押下時：対象設備の選択モードに遷移する
void OnDemolishPressed()

// 種付け指令ボタン押下時：対象牛の選択モードに遷移する
void OnBreedPressed()

// 搾乳指令ボタン押下時：対象牛の選択モードに遷移する
void OnMilkPressed()
```

---

## PriorityMenuUI : MonoBehaviour

ワーカーごとのタスク優先度を表示・編集するメニュー。

### プロパティ

| 名前 | 型 | 説明 |
|---|---|---|
| `TargetWorker` | `WorkerBase` | 現在表示中のワーカー |

### メソッド

```csharp
// 指定ワーカーのタスク優先度一覧を表示・更新する
void RefreshList(WorkerBase worker)

// タスク種別の優先度を変更する
// AIDecisionMaker.SetTaskPriority()に委譲する
void SetPriority(Type taskType, int priority)
```

## FacilityMenuUI : MonoBehaviour

施設配置モードへの入口となるメニュー。設備種別のボタン一覧を表示する。

### メソッド

```csharp
// 指定設備の配置モードを開始する
// BuildModeUI.EnterBuildMode()に委譲する
void OnFacilitySelected(EquipmentType type)
```

---

## FacilityPopupUI : MonoBehaviour

施設をクリックしたときに表示されるポップアップ。設備の情報を表示する。

### メソッド

```csharp
// 指定設備の情報を表示する
// 設備名・コスト・ムード補正値・現在状態を表示する
void Show(EquipmentBase equipment)

// ポップアップを非表示にする
void Hide()
```

---

## WorkerPopupUI : MonoBehaviour

ワーカーをクリックしたときに表示されるポップアップ。牛・牧場主の情報を表示する。

### メソッド

```csharp
// 指定ワーカーの情報を表示する
// 名前・ステータス（空腹度・スタミナ・幸福度）・状態異常・現在タスクを表示する
void Show(WorkerBase worker)

// ポップアップを非表示にする
void Hide()
```

---

## BuildModeUI : MonoBehaviour

施設配置モード中のUI。GhostPlacerと連携して配置プレビューを管理する。

### プロパティ

| 名前 | 型 | 説明 |
|---|---|---|
| `GhostPlacer` | `GhostPlacer` | ゴースト表示コンポーネントの参照 |
| `IsInBuildMode` | `bool` | 配置モード中かどうか |
| `CurrentType` | `EquipmentType` | 配置中の設備種別 |

### メソッド

```csharp
// 指定設備の配置モードを開始する
// GhostPlacer.ShowGhost()でプレビューを開始する
void EnterBuildMode(EquipmentType type)

// 配置モードを終了する
// GhostPlacer.HideGhost()でプレビューを終了する
void ExitBuildMode()

// プレイヤーがグリッドをクリックしたときの処理
// BuildingManager.PlaceEquipment()に委譲する
void OnGridClicked(Vector2Int position)
```

## TimeDisplayUI : MonoBehaviour

画面右下の時刻・倍率表示エリア。

### メソッド

```csharp
// 時刻表示と倍率表示を更新する
// GameTimeManager.OnSlotChanged / OnTickAdvanced に登録して使用する
void UpdateDisplay(int currentSlot, float timeScale)

// 一時停止ボタン押下時の処理
// GameTimeManager.SetPaused()に委譲する
void OnPausePressed()

// 速度変更ボタン押下時の処理
// GameTimeManager.SetTimeScale()に委譲する
void OnSpeedChanged(float scale)
```
