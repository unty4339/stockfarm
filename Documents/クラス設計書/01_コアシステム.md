## GameTimeManager

```
MonoBehaviour / Singleton
```

ゲーム内時間を管理する中枢クラス。1日=600tick、1tick=1秒（×1倍速）。

### プロパティ

| 名前 | 型 | 説明 |
|---|---|---|
| `Instance` | `GameTimeManager` | static シングルトン参照 |
| `CurrentTick` | `int` | 現在のtick（0〜599） |
| `CurrentDay` | `int` | 経過日数（0始まり） |
| `CurrentSlot` | `int` | 現在の時間スロット（0〜23、1スロット=25tick） |
| `TimeScale` | `float` | 時間倍率（1〜3） |
| `IsPaused` | `bool` | 一時停止フラグ |

### メソッド

```csharp
// 時間倍率を設定する（1〜3の範囲にclamp）
void SetTimeScale(float scale)

// 一時停止・再開を切り替える
void SetPaused(bool paused)
```

### イベント

```csharp
// tickが進むたびに発火（引数: currentTick）
event Action<int> OnTickAdvanced

// スロットが変わるたびに発火（引数: currentSlot）
event Action<int> OnSlotChanged

// 日付が変わるたびに発火（引数: currentDay）
event Action<int> OnDayChanged
```

### 時間スロットの対応表

| スロット | 時刻帯 | 用途 |
|---|---|---|
| 0〜3 | 深夜 | 夜（Sleep帯） |
| 4〜7 | 朝 | 朝（Work帯開始） |
| 8〜19 | 昼 | 昼（主作業帯） |
| 20〜23 | 夜 | 夜（Sleep帯） |

## ScheduleSlotType (enum)

1日24スロットそれぞれに割り当てるスケジュール種別。

```csharp
enum ScheduleSlotType
{
    Work,   // 労働
    Joy,    // 娯楽
    Sleep,  // 就寝
}
```
