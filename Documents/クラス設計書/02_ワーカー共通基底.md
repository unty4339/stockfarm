## WorkerBase

```
abstract MonoBehaviour
サブクラス: CowWorker, FarmerWorker
```

牛・牧場主に共通するステータス・状態異常・スケジュール管理の基底クラス。

### プロパティ

| 名前 | 型 | 説明 |
|---|---|---|
| `Hunger` | `float` | 空腹度（0〜100、0が満腹、100が飢餓） |
| `Stamina` | `float` | スタミナ（0〜100） |
| `Happiness` | `float` | 幸福度（0〜100） |
| `ActiveEffects` | `HashSet<StatusEffectType>` | 現在有効な状態異常の集合 |
| `Schedule` | `ScheduleSlotType[]` | 24スロット分のスケジュール配列 |
| `CurrentTask` | `AITaskBase` | 現在実行中のタスク（nullなら待機） |
| `GridPosition` | `Vector2Int` | グリッド上の現在位置 |

### ステータス更新メソッド

```csharp
// 空腹度をdeltaだけ変化させる（0〜100にclamp）
void UpdateHunger(float delta)

// スタミナをdeltaだけ変化させる（0〜100にclamp）
void UpdateStamina(float delta)

// 幸福度をdeltaだけ変化させる（0〜100にclamp）
void UpdateHappiness(float delta)
```

### 状態異常メソッド

```csharp
// 状態異常を付与する（重複付与は無視）
void ApplyEffect(StatusEffectType type)

// 状態異常を除去する
void RemoveEffect(StatusEffectType type)

// 指定の状態異常を持っているか判定する
bool HasEffect(StatusEffectType type)
```

### 計算メソッド

```csharp
// 作業効率を返す: Happiness/100 × (1 + workMood/100)
// workMoodは呼び出し元がRoomDetectorから取得して渡す
float GetWorkEfficiency(float workMood)

// スタミナ回復速度を返す: 基本値 × ベッド係数 × (1 + restMood/100)
// restMoodは呼び出し元がRoomDetectorから取得して渡す
float GetStaminaRecoveryRate(float restMood, float bedCoefficient)
```

### スケジュール操作メソッド

```csharp
// 指定スロットのスケジュール種別を取得する
ScheduleSlotType GetScheduleAt(int slot)

// 指定スロットのスケジュール種別を設定する
void SetScheduleAt(int slot, ScheduleSlotType type)

// 指定スロット範囲をまとめて設定する
void SetScheduleRange(int slotFrom, int slotTo, ScheduleSlotType type)
```

## StatusEffectType (enum)

ワーカーに付与される状態異常の種別。複数同時に重複可能。

```csharp
enum StatusEffectType
{
    Fatigue,          // 疲労
    Malnutrition,     // 栄養失調
    Hungry,           // 空腹
    PregnancyEarly,   // 妊娠（前期）    ※牛のみ
    PregnancyLate,    // 妊娠（後期）    ※牛のみ
    Lactating,        // 泌乳期          ※牛のみ
    Estrus,           // 発情期          ※牛のみ
}
```

### 状態異常の付与・除去条件概要

| 状態異常 | 付与条件 | 除去条件 |
|---|---|---|
| Fatigue | スタミナ 0 | スタミナ回復 |
| Malnutrition | Hungry 状態が継続 | 摂食 |
| Hungry | Hunger ≥ 閾値 | 摂食 |
| PregnancyEarly | 種付け成功から一定tick | PregnancyLate移行時 |
| PregnancyLate | PregnancyEarly 経過後 | 出産時 |
| Lactating | 出産後 | 乳蓄積量0かつ泌乳期間終了 |
| Estrus | 周期的に発生 | 種付け or 期間終了 |
