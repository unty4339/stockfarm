## EconomyManager : MonoBehaviour (Singleton)

資金の管理と資源売却を担うクラス。

### プロパティ

| 名前 | 型 | 説明 |
|---|---|---|
| `CurrentFunds` | `int` | 現在の資金量 |

### メソッド

```csharp
// 資金を加算する
void AddFunds(int amount)

// 資金を消費する（残高不足の場合はfalseを返す）
bool TrySpendFunds(int amount)

// 指定リソースリストを売却し、資金に換算する
// 各リソースの売却価格は GetSellPrice() で計算する
void SellResources(List<ResourceBase> resources)

// リソースの売却価格を返す
// 瓶牛乳: 40 × (Quality/100)
// バター: 80 × (Quality/100)
// チーズ: 120 × (Quality/100)
// Quality を持たない固体リソースは基本価格をそのまま返す
int GetSellPrice(ResourceBase resource)
```

### イベント

```csharp
// 資金が変化したときに発火（引数: 変化後の残高）
event Action<int> OnFundsChanged
```

### 初期値

| 項目 | 値 |
|---|---|
| 初期資金 | 2,000 |

## MapManager : MonoBehaviour (Singleton)

グリッドマップの状態管理と拡張を担うクラス。

### プロパティ

| 名前 | 型 | 説明 |
|---|---|---|
| `Width` | `int` | 現在のマップ幅（初期: 30） |
| `Height` | `int` | 現在のマップ高（初期: 30） |
| `Tiles` | `GridTile[,]` | グリッドタイルの二次元配列 |

### メソッド

```csharp
// 指定方向にマップを拡張する
// 拡張費用: 500 × 3^(N-1)（Nはその方向の拡張回数）
// EconomyManager.TrySpendFunds()で資金消費し、成功した場合のみ拡張する
bool TryExpand(MapExpandDirection direction)

// 指定方向の次回拡張費用を返す
int GetExpansionCost(MapExpandDirection direction)

// 指定座標のGridTileを返す（範囲外の場合は例外）
GridTile GetTile(Vector2Int position)

// 指定座標がマップ範囲内かを返す
bool IsValidPosition(Vector2Int position)
```

### 初期状態

| 項目 | 値 |
|---|---|
| 初期サイズ | 30×30 |
| 拡張可能方向 | 南・北・西 |
| 1回の拡張量 | 15マス |

```
class（データオブジェクト）
```

グリッド上の1マスを表すデータクラス。

### プロパティ

| 名前 | 型 | 説明 |
