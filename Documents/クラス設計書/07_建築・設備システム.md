## EquipmentBase

```
abstract MonoBehaviour
```

全設備の基底クラス。配置・撤去・ムード補正を共通管理する。

### プロパティ

| 名前 | 型 | 説明 |
|---|---|---|
| `Type` | `EquipmentType` | 設備種別 |
| `GridPosition` | `Vector2Int` | グリッド上の配置座標（左上基点） |
| `Size` | `Vector2Int` | 占有グリッドサイズ（例: 1×1, 2×2） |
| `BuildCost` | `int` | 建設コスト（資金） |
| `MoodBonus` | `int` | ムード補正値（正: 上昇、負: 低下） |
| `AffectedMoodType` | `MoodType` | 補正対象のムード種別 |
| `IsPlaced` | `bool` | 現在設置済みかどうか |

### メソッド

```csharp
// 指定座標に設置する（BuildingManagerから呼ばれる）
void Place(Vector2Int position)

// 設置を解除し、BuildCostの50%をEconomyManagerに返還する
void Remove()

// 設備が占有しているグリッド座標の一覧を返す
List<Vector2Int> GetOccupiedPositions()
```

## ProcessingEquipmentBase : EquipmentBase

```
abstract : EquipmentBase
```

加工設備の共通基底クラス。1サイクルの入出力とtick管理を担う。

### プロパティ

| 名前 | 型 | 説明 |
|---|---|---|
| `InputType` | `ResourceType` | 入力リソース種別 |
| `InputAmount` | `int` | 1サイクルあたりの入力量 |
| `OutputType` | `ResourceType` | 出力リソース種別 |
| `OutputAmount` | `int` | 1サイクルあたりの出力量 |
| `CycleTicks` | `int` | 1サイクルの所要tick数 |
| `IsProcessing` | `bool` | 加工中フラグ |
| `AssistantWorker` | `WorkerBase` | 加工補助中のワーカー（nullなら補助なし） |
| `InputTank` | `LiquidTank` | 入力用タンク（液体リソース使用設備のみ） |
| `OutputTank` | `LiquidTank` | 出力用タンク（液体リソース使用設備のみ） |

### メソッド

```csharp
// 加工サイクルを開始する
// assistantWorkerがnullでない場合、GetWorkEfficiency()を効率に乗算する
void StartCycle(WorkerBase assistantWorker)

// tickごとに進捗を更新する（GameTimeManager.OnTickAdvancedに登録）
void OnTick(int currentTick)
```

### イベント

```csharp
// 1サイクルが完了したときに発火
event Action OnCycleCompleted
```

### 加工設備の1サイクル仕様

| 設備 | 入力 | 入力量 | 出力 | 出力量 | 所要tick |
|---|---|---|---|---|---|
| ManualMilkingStand | - | - | Milk | 搾乳量 | - |
| AutoMilkingStand | - | - | Milk | 搾乳量 | 自動 |
| Churn（足踏み式攪拌機） | Milk | 10 | Butter | 1 | 100 |
| CheesePress（圧搾プレス機） | Milk | 20 | Cheese | 1 | 150 |
| BottleFiller（ボトル詰め機） | Milk | 5 | BottledMilk | 1 | 50 |

## 各設備の個別仕様

### FeedingTrough : EquipmentBase

給餌桶。牛が食料を消費する設備。

```csharp
// プロパティ
int CurrentFood    // 現在の食料残量
int MaxFood        // 最大食料量
bool AutoReload    // 自動補充フラグ（Chestから自動運搬タスクを生成）

// 食料を補充する（MaxFoodを超えない範囲で加算）
void Refill(int amount)

// 牛が食料を消費する（残量不足の場合はfalseを返す）
bool TryConsume(int amount)
```

### FoodShelf : EquipmentBase

食料棚。牧場主が栄養ペレットを取得する設備。

```csharp
// プロパティ
int StoredAmount  // 現在の保管量

// 食料を補充する
void Refill(int amount)

// 牧場主が食料を取得する（残量不足の場合はfalseを返す）
bool TryTake(int amount, out SolidResource food)
```

### AutoBrush : EquipmentBase

自動ブラシ機。牛のケアタスク対象設備。幸福度を向上させる。

```csharp
// プロパティ
bool IsInUse  // 使用中フラグ（1頭ずつ利用）

// ケアを実行し、ownerのHappinessを増加させる
void PerformCare(CowWorker owner)
```

### BreedingFacility : EquipmentBase

種付け施設。繁殖の実行起点となる設備。

```csharp
// プロパティ
bool IsInUse  // 使用中フラグ

// 繁殖を試みる
// successRate = cow.GetBreedSuccessRate(breedMood)
// 成功すれば cow.StartPregnancy() を呼ぶ
bool TryBreed(CowWorker cow, float breedMood)
```

### CowBed : EquipmentBase (abstract)

牛用ベッドの基底クラス。牧場主も使用可能（FarmerWorkerにも割り当て可）。

```csharp
// プロパティ
float StaminaRecoveryMultiplier  // 睡眠時のスタミナ回復倍率
WorkerBase AssignedWorker        // 割り当て済みワーカー（nullなら空き）

// ワーカーを割り当てる（既に割り当てがある場合は例外）
void Assign(WorkerBase worker)

// 割り当てを解除する
void Unassign()

// 指定ワーカーに割り当てられているかを返す
bool IsAssignedTo(WorkerBase worker)
```

### StrawBed / NormalBed / LuxuryBed / KingBed : CowBed

| クラス | サイズ | StaminaRecoveryMultiplier |
|---|---|---|
| `StrawBed` | 1×2 | 0.9 |
| `NormalBed` | 1×2 | 1.0 |
| `LuxuryBed` | 2×2 | 1.1 |
| `KingBed` | 3×3 | 1.1 |

いずれも `BuildCost = 200`、`MoodBonus = 20`（休憩ムード）、`AffectedMoodType = Rest`

## GhostPlacer : MonoBehaviour

施設配置モード中のプレビュー（ゴースト）表示を担うクラス。

```csharp
// 指定設備種別・座標にゴーストを表示する
void ShowGhost(EquipmentType type, Vector2Int position)

// ゴーストを非表示にする
void HideGhost()

// 指定位置に設備を配置可能かを判定する
// 他設備との重複・範囲外をチェックする
bool CanPlace(EquipmentType type, Vector2Int position)
```

## BuildingManager : MonoBehaviour (Singleton)

設備の設置・撤去を一元管理するシングルトン。

```csharp
// 指定設備を設置する
// TrySpendFunds()で資金消費し、成功した場合のみ設置する
void PlaceEquipment(EquipmentType type, Vector2Int position)

// 指定座標の設備を撤去する（50%返金）
void RemoveEquipment(Vector2Int position)

// 指定座標に設置されている設備を返す（なければnull）
EquipmentBase GetEquipment(Vector2Int position)

// 指定部屋内に含まれる全設備を返す
List<EquipmentBase> GetEquipmentsInRoom(RoomData room)

// 全設置済み設備を返す
List<EquipmentBase> GetAllEquipments()
```
